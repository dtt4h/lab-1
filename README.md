# Лабораторная работа №1 - SPA для гостиницы

**Выполнил:** студент группы [Твоя группа]  
**Тема:** Система учета инцидентов безопасности (Гостиница "Космос")

## Что сделано

1. **Главная страница (Инциденты)**
   - Таблица всех инцидентов с сортировкой по дате (новые сверху)
   - Поиск по названию, статусу, гостинице
   - Пагинация (по 5 записей на странице)
   - Подтверждение перед удалением
   - Статусы: Открыт / Проверяется / Решён

2. **Страница сотрудников**
   - Таблица с ФИО, должностью, сменой, телефоном
   - Поиск по имени или должности
   - Пагинация
   - Возможность добавлять/редактировать/удалять

3. **Формы**
   - Добавление/редактирование инцидентов (дата, этаж, уровень, статус)
   - Добавление сотрудников (смена: утро/день/вечер/ночь)
   - Простая валидация (alert если не заполнено)

4. **Данные**
   - Хранятся в `db.json` (json-server)
   - 20 тестовых инцидентов
   - 16 сотрудников
